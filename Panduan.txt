Panduan Instalasi dan Menjalankan Aplikasi Python

1. Buka Command Prompt (CMD) di Folder Proyek  
   Pastikan berada di direktori proyek yang sesuai. Bisa melakukannya dengan membuka Command Prompt, lalu navigasikan ke folder proyek menggunakan perintah berikut:  
   
   cd path/to/folder/proyek
   

2. Menginstal Dependensi 
   Jalankan perintah berikut untuk menginstal semua dependensi yang dibutuhkan oleh aplikasi. Pastikan file `requirements.txt` ada di direktori proyek.  
   
   pip install -r requirements.txt
   

3. Menjalankan Aplikasi 
   Setelah semua dependensi terinstal, jalankan server aplikasi menggunakan perintah:  
  
   py manage.py runserver
